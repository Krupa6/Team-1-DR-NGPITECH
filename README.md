# AI Future Campus

An AI-powered career guidance and employability platform that helps students prepare for an AI-driven future through skill assessment, personalized roadmaps, project recommendations, and mentor guidance.

🚀 Overview

AI Future Campus enables students to:

Assess their technical and aptitude skills
Choose a desired career path
Upload project portfolios
Receive AI-generated skill analysis
Get classified into skill levels
Follow personalized learning roadmaps
Receive project recommendations
Connect with suitable mentors
Track progress through an interactive dashboard

# 🎯 Problem Statement

Students often struggle to:

Identify their current skill level
Understand industry expectations
Choose the right learning path
Build relevant projects
Find suitable mentors
Stay aligned with rapidly evolving AI-driven careers
